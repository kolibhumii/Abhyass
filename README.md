# Abhyass

Abhyass is a web-based assignment submission tracker designed to help students and educators manage academic tasks efficiently. It provides a secure and structured platform for handling coursework.

## Overview

The system simplifies assignment submission, storage, and tracking using a cloud-based backend and a clean user interface.

## Features

- User authentication using Firebase Authentication  
- Assignment submission and tracking  
- Dashboard for organized data viewing  
- Real-time database integration using Firestore  
- Instant data synchronization  

## Tech Stack

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Firebase Authentication  
- Firebase Firestore Database  

## Installation

1. Clone the repository:
    git clone https://github.com/kolibhumii/Abhyass.git
    cd Abhyass
2. Configure Firebase:
- Create a Firebase project  
- Enable Authentication  
- Enable Firestore Database  
- Add your config in `firebase.js`

3. Run:
- Open `index.html` in a browser
